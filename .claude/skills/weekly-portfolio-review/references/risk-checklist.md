# 持仓风险全面检查清单

> 每次周复盘时，对每只持仓快速扫描以下清单。
> 触发任意一条「🚨 严重」需立即深入研究，不能拖到下周。

---

## 公司基本面风险

### 财务健康
- [ ] 最新季报是否出现营收/利润大幅 miss（超 5%）？
- [ ] 管理层是否下调全年指引（Guidance Cut）？
- [ ] 自由现金流是否连续两季度为负（非早期成长股）？
- [ ] 应收账款增速是否远超营收增速（可能虚增收入）？
- [ ] 库存是否大幅积压（制造/消费行业）？

🚨 严重：
- [ ] 审计师出具非标准意见或"持续经营"疑虑
- [ ] 财务重述（Restatement）
- [ ] CFO 突然离职（非计划性）

### 竞争护城河
- [ ] 核心产品/服务市场份额是否在下滑？
- [ ] 竞争对手是否推出了实质性替代产品？
- [ ] 客户流失率（Churn）是否上升？
- [ ] 定价权是否受到压力（毛利率连续下滑）？

🚨 严重：
- [ ] 最大客户（占收入 > 15%）宣布转向竞争对手
- [ ] 核心专利到期且无后续产品线

### 管理层与治理
- [ ] CEO / CFO 是否有非计划性变动？
- [ ] 是否有高管大规模非计划性减持？
- [ ] 董事会是否出现重大争议？
- [ ] 是否有 SEC 调查或重大诉讼新进展？

---

## 估值风险

- [ ] 股价上涨后 P/E 是否已超过历史均值的 1.5 倍？
- [ ] P/S 是否已远超同行业均值（成长股适用）？
- [ ] DCF 估值下，当前股价隐含的增速是否过于乐观？
- [ ] 分析师目标价是否已被股价大幅超越？

🚨 严重：
- [ ] 股价在无基本面支撑的情况下短期暴涨 50%+（FOMO 驱动）
- [ ] 散户/社交媒体热度异常高涨（meme 化风险）

---

## 宏观与行业风险

### 利率与宏观
- [ ] 若持有成长股：利率预期是否上升（压缩估值）？
- [ ] 若持有消费股：消费信心指数是否持续下滑？
- [ ] 若持有能源股：油价趋势是否逆转？
- [ ] 是否有直接受关税/贸易政策影响的业务？

### 行业特定
- [ ] 行业是否进入监管收紧周期？
- [ ] 行业是否出现颠覆性技术（AI 替代等）？
- [ ] 供应链是否面临新的中断风险？

---

## 仓位风险

- [ ] 单只股票仓位是否因上涨超过 15%？
- [ ] 是否有多只高度相关的持仓（同涨同跌）？
- [ ] 加总高 beta 仓位，整体组合波动性是否超出承受范围？
- [ ] 亏损持仓是否因沉没成本心理而迟迟未止损？

🚨 严重：
- [ ] 单只股票仓位超过 25%
- [ ] 账户总亏损超过预设的最大回撤线

---

## 风险评级汇总

完成以上检查后，给每只持仓打一个综合风险评级：

```
低风险：无任何触发项
中风险：触发 1-2 个普通项（非🚨）
高风险：触发 3 个以上普通项，或任意 1 个🚨项
极高风险：触发 2 个以上🚨项
```

对应建议：
```
低风险   → 继续持有，正常观察
中风险   → 减仓观察，等待下季度财报验证
高风险   → 认真考虑清仓，至少大幅减仓
极高风险 → 立即评估是否清仓，不拖延
```
