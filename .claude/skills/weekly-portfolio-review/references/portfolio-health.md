# 组合健康度评估框架

---

## 一、分散度评估

### 行业分散
```
健康组合的行业分布参考：

单一行业占比     评级
> 50%           🚨 过度集中，行业风险极高
30% - 50%       ⚠️ 偏高，注意行业周期风险
15% - 30%       ✅ 适中
< 15%           ✅ 充分分散

常见行业代码（GICS）：
  科技（Technology）
  通信服务（Communication Services）
  消费可选（Consumer Discretionary）
  消费必需（Consumer Staples）
  医疗健康（Health Care）
  金融（Financials）
  工业（Industrials）
  能源（Energy）
  公用事业（Utilities）
  房地产（Real Estate）
  材料（Materials）
```

### 风格分散
```
成长 vs 价值 vs 防御 的合理比例（参考，因人而异）：

激进型投资者（高风险承受）：
  成长股 60-70% / 价值股 20-30% / 防御股 0-10%

平衡型投资者：
  成长股 40-50% / 价值股 30-40% / 防御股 10-20%

保守型投资者：
  成长股 20-30% / 价值股 30-40% / 防御股 30-40%

判断标准：
  成长股 = 营收增速 > 15%，高 P/S 或 P/E
  价值股 = 低于历史估值，稳定盈利，股息
  防御股 = 必需消费/公用事业/医疗，低 beta
```

### 持仓数量
```
持仓数量     特点
< 5 只      高集中，高风险高回报，需要极深研究
5 - 10 只   集中型，主动管理者常见选择
10 - 20 只  适中，平衡研究深度和分散
20 - 30 只  偏分散，接近指数化
> 30 只     高度分散，个股影响减弱，不如直接买 ETF

新手建议：5-10 只，研究深度 > 分散数量
```

## 二、相关性评估

高相关性持仓列举（需警惕）：
```
以下组合实际上是「同一个赌注」：
  NVDA + AMD + INTC = 都是半导体，高度相关
  AAPL + MSFT + GOOGL = 大盘科技，共同承受估值压缩
  XOM + CVX = 都跟油价走
  
打破相关性的方法：
  ① 跨行业配置（科技 + 消费 + 医疗）
  ② 跨市值配置（大盘 + 中盘）
  ③ 加入真正防御性标的（T、JNJ 等低 beta 股）
  ④ 保留一定现金（现金与股票相关性 = 0）
```

## 三、盈亏结构评估

```
健康的盈亏结构：
  ① 盈利仓位的总收益 > 亏损仓位的总亏损（组合整体盈利）
  ② 最大单仓亏损不超过账户总值的 5%
  ③ 没有「沉默的地雷」（明显逻辑失效但因亏损不愿承认的仓位）

危险信号：
  ① 亏损仓位越来越大（越跌越加仓，无逻辑支撑）
  ② 用盈利仓位的收益掩盖亏损仓位的损失（表面平衡实则集中风险）
  ③ 某只亏损股已经占到账户的 10%+（超出合理止损范围）
```

## 四、组合 Beta 评估

Beta 衡量你的组合相对大盘的波动幅度：
```
组合 Beta = Σ（单股 Beta × 仓位比例）

Beta 参考值：
  > 1.5   高波动，大盘跌 10% 你可能跌 15%+
  1.0 - 1.5  与大盘同步，略高波动
  0.7 - 1.0  略低波动，接近大盘
  < 0.7   低波动，防御性强

各类股票典型 Beta（参考）：
  高 Beta：科技成长股（NVDA ~1.8）、加密相关、小盘股
  中 Beta：消费可选、工业（1.0-1.4）
  低 Beta：必需消费、医疗、公用事业（0.4-0.8）

判断：当前组合 Beta 是否与你的风险承受能力匹配？
```

## 五、组合健康度评分卡

```
维度              评分（1-5）  说明
────────────────────────────────────────
行业分散度         ___         最大单行业占比 < 30% = 5分
风格平衡           ___         符合个人风险偏好 = 5分
持仓数量合理性     ___         5-15 只且每只研究充分 = 5分
相关性控制         ___         无高度相关的重复赌注 = 5分
盈亏结构健康       ___         无沉默地雷，亏损可控 = 5分
Beta 匹配度        ___         与风险承受能力一致 = 5分
────────────────────────────────────────
总分              ___/30

25-30分：组合健康，维持现状
18-24分：有优化空间，制定改进计划
< 18分：需要较大调整，优先处理最低分维度
```
