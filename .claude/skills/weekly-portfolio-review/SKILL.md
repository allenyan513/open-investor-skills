# 持仓周复盘 Skill

> ⚠️ 本 Skill 为持仓管理辅助工具，不构成投资建议。
> 所有「建议操作」仅供参考，最终决策需结合你的风险承受能力和税务情况。

## 持仓数据来源

**在开始复盘之前，先读取持仓文件。**

### 标准持仓文件

```
路径：{项目根目录}/portfolio/holdings.csv
```

**CSV 字段说明**：

| 字段 | 说明 | 是否必填 |
|------|------|---------|
| `ticker` | 股票代码（如 AAPL） | ✅ 必填 |
| `shares` | 持股数量 | ✅ 必填 |
| `avg_cost` | 持仓均价（USD） | ✅ 必填 |
| `target_price` | 目标价（USD） | 选填 |
| `stop_loss` | 止损价（USD） | 选填 |
| `position_pct` | 仓位占组合比例（%） | 建议填写 |

```
1. 复制 portfolio/holdings.example.csv → 重命名为 holdings.csv
2. 填入自己的持仓数据（可用 Excel / Numbers / 任意文本编辑器）
3. 每次持仓变动后手动更新（买入、卖出、加减仓）
4. 触发本 Skill 时无需手动粘贴，Claude 会自动读取该文件
```

## 核心理念

> **持有一只股票，等于每天在当前价格重新买入它。**
> 如果你今天不知道这只股票，你还会买吗？

周复盘的目的不是频繁交易，而是**防止「惰性持仓」**——
忘了当初为什么买，涨了不知道该不该卖，跌了不知道该不该加仓。

```
惰性持仓的危险信号：
  ✗ "反正放着吧，涨回来再说"
  ✗ "我已经亏了这么多，不想卖"（沉没成本陷阱）
  ✗ "这只我研究了很久，应该没问题的"（确认偏误）
  ✗ 上次认真看这只股票是三个月前

正确的持仓心态：
  ✓ 我知道我持有它的原因（买入论文）
  ✓ 我知道什么情况下该卖（失效条件）
  ✓ 我定期检查这些条件是否发生了变化
```

## 复盘流程：四个问题

对持仓中的**每一只股票**，依次回答这四个问题：

---

### 问题一：买入逻辑还成立吗？

这是最重要的问题。买入逻辑失效 = 无论盈亏都应该考虑退出。

```
常见买入逻辑类型及验证方式：

逻辑类型          核心验证点
────────────────────────────────────────────
成长逻辑          营收/用户增速是否维持？
  （买的是增长）   管理层指引是否被下调？

估值逻辑          当前估值是否仍低于内在价值？
  （买的是低估）   股价上涨后安全边际是否消失？

催化剂逻辑        催化剂是否已兑现？
  （买的是事件）   若已兑现，持续持有的理由是什么？

行业趋势逻辑      行业趋势是否仍然持续？
  （买的是赛道）   公司在行业中的地位是否变化？

护城河逻辑        护城河是否被侵蚀？
  （买的是壁垒）   竞争格局是否发生重大变化？
```

**搜索验证**（每只股票）：
```
[TICKER] latest news this week
[TICKER] earnings guidance update
[TICKER] competitive landscape change
```

**结论选项**：
- ✅ 逻辑完整成立 → 继续持有
- ⚠️ 逻辑部分弱化 → 减仓观察
- 🚨 逻辑已根本失效 → 认真考虑清仓

---

### 问题二：有没有出现新的风险？

买入之后，世界在变化。检查三类新风险：

**① 公司层面新风险**
```
- 最近是否有重大负面新闻（监管/诉讼/高管变动）？
- 最新财报是否出现红旗？（参考 earnings-report-decoder Skill）
- 竞争对手是否推出了有威胁的新产品？
- 是否有大规模裁员或业务收缩？
```

**② 估值层面新风险**
```
- 股价上涨后估值是否已经透支未来增长？
- P/E 或 P/S 是否已远超历史均值？
- 当前仓位占比是否因股价上涨而过度集中？
```

**③ 宏观/行业层面新风险**
```
- 利率环境是否对该股票不利（成长股对利率敏感）？
- 行业监管政策是否出现实质性变化？
- 该公司的主要客户/供应链是否面临压力？
```

读取 `references/risk-checklist.md` 获取完整的风险检查清单。

---

### 问题三：仓位比例是否合理？

持仓时间越长，仓位越容易失衡——赢家越来越大，输家越来越小（或反之）。

```
仓位检查框架：

单只股票占比       建议行动
───────────────────────────────
< 5%              正常，可继续持有
5% - 10%          正常，高波动股需关注
10% - 15%         偏高，评估是否需要修剪（trim）
> 15%             集中度过高，除非极高确信度
> 25%             🚨 单一持仓过大，建议评估风险

特殊情况：
  某只股票大涨导致仓位膨胀 → "幸福的烦恼"，需决定是否修剪锁利
  某只股票大跌导致仓位萎缩 → 结合问题一决定是否补仓或止损
```

**再平衡触发条件**：
```
① 单只股票仓位 > 原计划仓位的 2 倍
② 总持仓股票数量变化（买入新仓时，是否有旧仓需要腾出空间）
③ 整体组合与风险承受能力不符（如高成长股占比过高）
```

---

### 问题四：与买入时相比，性价比如何？

股价变化后，当前的风险/回报比可能已经改变。

```
四种持仓状态判断：

状态                股价变化    逻辑变化    建议
─────────────────────────────────────────────
逻辑成立 + 价格更低  下跌        成立        考虑加仓（如有余仓）
逻辑成立 + 价格更高  上涨        成立        持有，评估是否修剪
逻辑减弱 + 价格更低  下跌        弱化        止损，不要越跌越买
逻辑失效 + 价格更高  上涨        失效        清仓，不要恋战

最危险的陷阱：
  「逻辑减弱 + 价格更低 → 越跌越买」
  这是价值陷阱（Value Trap）的标准路径
  区分「暂时性下跌」和「逻辑根本破坏」
```

## 组合层面检查

完成个股复盘后，从整体组合视角再做一次检查：

```
① 行业集中度
   单一行业占总仓位 > 40% → 行业风险过于集中
   
② 风格集中度
   成长股 vs 价值股比例是否符合预期？
   高 beta vs 低 beta 的比例？

③ 现金比例
   当前现金/总资产：____%
   参考原则：市场估值偏高时保留更多现金（10-20%）
             看到明确机会时可降低现金（< 5%）

④ 最大持仓相关性
   前三大持仓是否高度相关（同涨同跌）？
   若是，实际分散效果有限
```

详细的组合健康度评估见 `references/portfolio-health.md`。

## 复盘节奏建议

```
周复盘（每周一次，15-30 分钟）
  → 扫描每只持仓的新闻和价格变化
  → 快速过四个问题
  → 有变化的仓位标记为「需深入研究」

月复盘（每月一次，1-2 小时）
  → 结合最新财报更新买入逻辑
  → 重新评估估值是否合理
  → 调整仓位比例
  → 使用 stock-fundamental-research Skill 深度更新研究

季度复盘（每季度，半天）
  → 财报季结束后系统性评估
  → 对每只持仓重跑完整的四步研究
  → 决定是否需要更换持仓
```

