# 市场新闻与情绪分析 Skill

> ⚠️ 本 Skill 为新闻解读辅助，不构成投资建议。
> 新闻情绪分析存在高度不确定性，仅供参考，请结合基本面综合判断。

---

## 核心理念：区分噪音与信号

市场每天产生大量新闻，**90% 是噪音，10% 是真正影响基本面的信号**。
本 Skill 的核心任务是帮你区分这两类，避免因噪音做出错误决策。

```
噪音（应忽略）          信号（需认真对待）
──────────────         ─────────────────
分析师目标价调整         核心业务指标变化
短期市场情绪波动         竞争格局根本改变
宏观恐慌情绪蔓延         管理层重大变动
社交媒体热度             监管政策实质性收紧
技术面突破/跌破          客户/合同重大得失
```

---

## 第一步：新闻分类

拿到新闻后，先归类到以下 7 种类型之一，不同类型有不同的分析框架：

| 类型 | 典型触发词 | 跳转框架 |
|------|-----------|---------|
| **A 财务业绩** | earnings、revenue、EPS、guidance、beat/miss | → 框架 A |
| **B 业务事件** | 合同、并购、产品发布、客户、合作 | → 框架 B |
| **C 监管政策** | SEC、antitrust、fine、regulation、ban、investigation | → 框架 C |
| **D 宏观经济** | Fed、interest rate、CPI、GDP、recession | → 框架 D |
| **E 行业动态** | 竞争对手、行业趋势、供应链、原材料 | → 框架 E |
| **F 高管人事** | CEO、CFO、resign、appoint、insider selling | → 框架 F |
| **G 市场情绪** | 评级调整、做空报告、社交媒体、散户热度 | → 框架 G |

---

## 框架 A：财务业绩类新闻

财报数字出来后，核心是判断「相对预期」而非绝对数字好坏。

```
三步判断法：

① 结果 vs 预期
   营收/EPS 超预期多少？（超 >5% = 大幅超预期）
   哪个指标 beat，哪个 miss？

② 指引 vs 预期（最重要！）
   下季度/全年指引是否超过分析师共识？
   "Guidance cut"（下调指引）= 🚨 最大负面信号
   "Raised guidance"（上调指引）= 最强正面信号

③ 核心 KPI 趋势
   非财务指标（用户数/订阅/ASP）是加速还是减速？
   毛利率趋势是扩张还是收缩？
```

**常见陷阱**：
- EPS beat 但主要靠回购减少股数，而非利润真正增长 → 质量差
- 营收 beat 但指引低于预期 → 市场通常仍然下跌
- 大幅 beat 但股价下跌 → 说明预期已提前定价，"buy the rumor, sell the news"

## 框架 B：业务事件类新闻

### 并购（M&A）
```
收购方（Acquirer）通常短期承压：
  - 溢价越高（>30%）→ 股价压力越大
  - 现金收购 vs 股票收购：股票收购稀释现有股东
  - 问：协同效应是否可信？整合风险？

被收购方（Target）通常大涨：
  - 溢价通常 20-50%
  - 关注：是否有其他竞标方（bidding war）？
  - 关注：监管审查风险（反垄断）？
```

### 重大合同/客户
```
利好程度判断：
  合同金额 / 年营收 > 10% → 重大利好 ✅
  合同金额 / 年营收 1-10% → 中等利好
  合同金额 / 年营收 < 1%  → 噪音，可忽略

注意：
  - 单一大客户依赖度增加 → 集中风险上升（双刃剑）
  - 政府合同通常毛利率较低但稳定
```

### 产品发布
```
判断维度：
  - 是否进入新市场（TAM 扩张）？
  - 是否对现有产品形成替代（自我蚕食）？
  - 竞争对手是否已有类似产品？
  - 发布时间线：即将量产 vs 概念阶段（差别巨大）
```

## 框架 C：监管政策类新闻

监管新闻是最难判断的，需区分实质影响和情绪冲击。

```
严重程度分级：

🔴 极高（立即实质影响）
   - 核心业务被明令禁止
   - 大额罚款（>净利润 10%）
   - 强制分拆或剥离核心资产
   - 高管被刑事起诉

🟠 较高（中期影响，需跟踪）
   - 正式立案调查（investigation opened）
   - 新法规要求大幅改变商业模式
   - 关键市场准入受限

🟡 中等（影响有限，情绪冲击为主）
   - 初步调查询问（request for information）
   - 行业整体性监管趋严（非针对单一公司）
   - 罚款金额可控（<净利润 5%）

🟢 低（基本是噪音）
   - 分析师/媒体猜测监管风险
   - 国会听证但无实质立法进展
   - 其他市场已经历类似监管（可预测）
```

**历史规律**：监管初期情绪冲击往往大于实际影响，
优质公司在监管情绪低点往往是买入机会（参考 2021 年中概股、2023 年银行业）。

## 框架 D：宏观经济类新闻

宏观新闻对个股的影响高度依赖行业属性。

```
Fed 加息/降息 对不同行业的影响：

加息（利率上升）：
  受损：科技成长股（估值压缩）、REITs（融资成本上升）、
        高负债企业、长久期债券
  受益：银行（NIM 扩张）、保险、短期债券

降息（利率下降）：
  受益：科技成长股（估值扩张）、REITs、消费贷款
  受损：银行（NIM 收窄）、保险投资收益

CPI 数据：
  高于预期 → 市场担心 Fed 继续加息 → 成长股承压
  低于预期 → 通胀受控 → 成长股反弹

GDP/就业数据：
  过热（超预期）→ Fed 鹰派 → 市场短期波动
  过冷（低预期）→ 衰退担忧 → 周期股承压，防御股受益
```

**重要原则**：宏观新闻产生的个股波动通常是短期情绪冲击，
基本面强的公司往往在宏观恐慌后快速修复。

## 框架 E：行业动态类新闻

```
竞争格局变化判断：
  竞争对手出重大问题（破产/召回/诉讼）
    → 市场份额转移机会 → 行业第二/第三受益

  行业新进入者（尤其是科技巨头）
    → 护城河被威胁程度取决于转换成本

  供应链中断
    → 短期：有库存的公司受益
    → 中期：依赖单一供应商的公司风险更高

行业级利好/利空传导：
  利好先看行业龙头，再扩散到小公司
  利空先看弱势公司，再传导到龙头（通常影响较小）
```

## 框架 F：高管人事类新闻

人事变动是市场最敏感的信号之一，原因是信息不对称：内部人知道的比我们多。

```
CEO 变动：
  突然辞职（sudden resignation）
    → 🚨 极度负面信号，通常预示内部问题
    → 问：是否伴随财务重述？SEC 调查？
  正常退休/接班计划
    → 中性，关注继任者背景
  知名外部空降（来自成功公司）
    → 通常正面，市场认可变革信号

CFO 变动：
  比 CEO 变动更值得警惕
  CFO 突然离职往往与财务问题相关

内部人买卖（Insider Transactions）：
  高管集体买入（open market purchase）
    → ✅ 强烈正面信号，他们用自己的钱背书
  高管集体卖出
    → 需分析原因：定期计划卖出（10b5-1）vs 主动决策
    → 定期计划卖出通常是噪音；多名高管同时卖出非计划股份 = 🚨

查询内部人交易：
  https://www.sec.gov/cgi-bin/browse-edgar?action=getcompany&CIK=[TICKER]&type=4
```

## 框架 G：市场情绪类新闻

这类新闻通常是噪音，但在极端情况下会自我强化。

```
分析师评级调整：
  升级（Upgrade）到 Buy/Outperform → 短期提振，但影响有限
  降级（Downgrade）到 Sell/Underperform → 短期压力
  目标价调整：关注幅度，>20% 的调整更有意义
  注意：分析师往往滞后，不要单独依赖

做空报告（Short Seller Report）：
  来源可信度：Hindenburg/Muddy Waters 历史准确率高，需认真对待
  普通做空报告：通常夸大，但可能引发调查
  应对：检查报告中的具体指控，逐条核实
  规律：股价往往在报告发布后过度下跌，基本面好的公司会反弹

社交媒体/散户热度（meme stocks）：
  🚨 高度警惕：散户驱动的上涨通常不可持续
  特征：成交量异常放大、期权未平仓合约激增、
        Reddit/X 上讨论量暴增
  策略：不追高；如已持有，需判断基本面是否支撑新估值
```

## 情绪评分系统

分析完新闻后，输出统一的情绪评分卡：

### 影响方向
```
+3  强烈利好（核心业务重大正面变化，指引上调）
+2  明显利好（财务超预期，重要合同，战略利好）
+1  轻微利好（小额利好，行业正面，情绪提振）
 0  中性/噪音（影响不明确或可忽略）
-1  轻微利空（小额不利，情绪冲击，短暂影响）
-2  明显利空（财务低于预期，指引下调，监管调查）
-3  强烈利空（核心业务受损，管理层危机，重大处罚）
```

### 持续时间预判
```
短暂（1-3 天）：纯情绪冲击，无基本面变化
中期（1-4 周）：需要数据验证的事件
长期（1 季度+）：实质性基本面变化
永久性：竞争格局或商业模式根本改变
```

### 置信度
```
高   明确数字支撑，事实确认
中   信息部分，需跟进验证
低   传言/分析师猜测/情绪驱动
```

## 批量新闻监控模式

当用户提供多条新闻（或说"帮我看看今天的消息"）时，切换为**快速扫描模式**：

```
每条新闻输出简版评分：

1. [新闻标题]
   → 类型：[字母] | 评分：[±数字] | 持续：[时间] | 一句话：[判断]

2. [新闻标题]
   → 类型：[字母] | 评分：[±数字] | 持续：[时间] | 一句话：[判断]

---
今日总体情绪：[偏多 / 中性 / 偏空]
最值得关注的 1 条：[标题] → [理由]
```

## 报告保存

**分析完成后执行：生成 MD → 保存**

### 文件命名规则
```
[TICKER]_news_[YYYYMMDD_HHMM].md / .pdf
示例：NVDA_news_20250314_1430.md
保存路径：当前目录下的 output/ 子目录
```

**执行顺序**：
```
[分析完成]
     ↓
1. mkdir -p output
2. 写入 output/[TICKER]_news_[DATE_TIME].md
     ↓
"✅ 分析完成 | 评分：[±X] | 文件：output/... | 邮件：已发送"
```
