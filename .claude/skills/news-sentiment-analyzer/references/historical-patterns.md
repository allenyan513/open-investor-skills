# 历史同类新闻的市场反应规律

> 以下规律基于历史统计，不保证未来重复。用于校准预期，而非预测。

---

## 财报季规律

### Beat & Raise（超预期 + 上调指引）
```
历史表现：
  发布当天：平均 +5% 到 +12%（取决于超预期幅度）
  后续 1 个月：若基本面支撑，通常继续上行
  规律：超预期幅度越大，且指引越乐观，涨幅越持久

注意事项：
  "Buy the rumor, sell the news"：若股价已提前大涨 20%+，
  即使超预期也可能当天下跌（期望已定价）
```

### Miss & Cut（低于预期 + 下调指引）
```
历史表现：
  发布当天：平均 -8% 到 -20%（成长股惩罚更重）
  后续走势：通常需要 1-2 个季度才能企稳

严重程度分级：
  仅 Miss（无 Guidance Cut）→ 短期冲击，1-2 周企稳
  Miss + Guidance Cut → 中期下行压力，需等下季度验证
  连续 2 次 Miss → 估值体系重构，可能长期承压
```

### Guidance Cut without Miss（利润达标但下调指引）
```
市场反应通常比 Miss 更负面，因为：
  - 说明管理层对未来看法悲观
  - 远期盈利预测需要全面下调
历史表现：平均 -10% 到 -15%
```

## 并购事件规律

### 大型收购宣布当天
```
收购方平均：-2% 到 -5%（市场对整合风险和溢价的折扣）
被收购方平均：+20% 到 +35%（接近收购溢价）

影响收购方跌幅的因素：
  溢价高低（>40% 溢价 → 收购方跌更多）
  股票 vs 现金支付（股票支付 → 额外稀释担忧）
  战略逻辑是否清晰
  收购方本身估值水平（已贵的公司更脆弱）
```

### 收购失败/被监管阻止
```
收购方：通常反弹（市场认为避免了"坏收购"）
被收购方：大幅下跌，通常跌回原价附近（失去溢价）
```

## 监管事件规律

### 初次调查宣布
```
历史平均当天跌幅：-5% 到 -15%
后续规律：
  多数调查最终以和解结束（罚款金额通常可控）
  过度反应后往往是买入机会（前提：基本面完好）

例外情况：
  若调查涉及核心商业模式 → 长期压制
  若调查同时伴随财务重述 → 🚨 非常严重
```

### 大额罚款确认
```
市场通常"预期折扣"：若罚款低于市场预期 → 股价反而上涨
规律：一旦法律不确定性消除，即使坏消息也可能是利好（确定性溢价）
```

## 高管变动规律

### CEO 突然辞职
```
历史当天跌幅：-5% 到 -25%（高度不确定，取决于原因）
后续走势：
  若原因是"个人原因/健康"：市场通常给 1-2 周观察期
  若原因涉及财务不当：长期压制，可能引发更多调查
  若新 CEO 是知名人物：可能快速企稳甚至反弹
```

### 知名投资人/活动股东建仓
```
巴菲特/伯克希尔披露建仓（13F）→ 平均 +5% 即时反应
激进投资者（activist）建仓通知 → 平均 +10%，
  但后续高度依赖其推动变革的能力
```

## 宏观数据事件规律

### 非农就业（NFP）
```
热于预期（就业强劲）→ 短期：Fed 鹰派预期 → 成长股跌
冷于预期（就业疲软）→ 短期：衰退担忧 vs 降息预期，方向不确定
"完美"数据（温和增长）→ 通常推动股市上涨
```

### CPI 发布
```
高于预期（通胀超预期）→ 成长股平均 -1% 到 -3%
低于预期（通胀缓和）→ 成长股平均 +1% 到 +4%
影响在发布后 2 小时内最大，通常当天收敛
```

### Fed 会议（FOMC）
```
加息 25bp（已预期）→ 市场平静
加息 50bp（超预期）→ 成长股 -3% 到 -8%
降息（未预期）→ 通常大涨，但也可能引发"经济衰退"担忧

规律：预期内的行动 = 中性；超预期的行动 = 大幅波动
      关注鲍威尔发布会用词，比加息本身更重要
```

## "反直觉"规律汇总

这些规律违反直觉，但在市场中反复出现：

```
1. 坏消息出清后的反弹
   长期拖累的不确定性（调查、诉讼）一旦以确定的方式结束，
   即使结果不好，股价也可能上涨（不确定性溢价消除）

2. 好财报后下跌
   "Buy the rumor, sell the news"
   若股价已提前涨了 30%，即使财报超预期也可能当天跌
   判断标准：财报前股价是否已经大涨？

3. 裁员公告后上涨
   裁员意味着成本控制，若市场认为公司"臃肿"，
   大规模裁员可能被解读为积极的效率提升信号

4. 降级后的短暂买点
   分析师降级通常滞后于股价下跌
   降级后第二天有时是短期底部（情绪充分释放）

5. 行业利空但个股上涨
   行业性利空会打压所有公司，但市场份额领导者
   往往在竞争对手出问题后获得更多关注和信任
```
