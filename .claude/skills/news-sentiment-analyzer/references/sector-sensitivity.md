## 行业专属敏感点详解

### 科技/SaaS
**最敏感**：用户增长放缓、NRR（净收入留存率）下降、竞争对手定价战
**利率敏感原因**：高成长股估值用 DCF，利率上升直接压缩远期现金流现值
**监管风险**：数据隐私（GDPR/CCPA）、反垄断（尤其是大型平台）
**特殊信号**：
- 企业客户预算收缩消息 → SaaS 整体承压
- AI 概念新进展 → 相关股票情绪大幅波动（但基本面影响需数季度验证）

### 半导体
**最敏感**：出口管制（中国市场占比大）、下游需求预警、库存周期
**周期特征**：半导体是高度周期性行业，同一新闻在周期不同位置影响截然不同
**供应链传导**：台积电/ASML 的任何产能消息会直接传导到整个产业链
**特殊信号**：
- PC/手机出货量数据 → 消费级芯片需求先行指标
- 数据中心资本支出（AWS/Azure/Google CapEx）→ AI 芯片需求指标

### 金融/银行
**最敏感**：利率变化（直接影响 NIM）、信贷质量（NPL 上升）
**系统性风险特征**：单一银行的流动性问题会快速传染（参考 2023 SVB 事件）
**特殊信号**：
- 存款流失（deposit outflows）→ 🚨 流动性风险
- 贷款损失准备金（loan loss provisions）大幅增加 → 预期信贷恶化
- Fed 压力测试结果 → 影响分红和回购能力

### 医疗/生物科技
**最敏感**：FDA 审批结果（二元事件，成或败）、临床数据
**二元事件特征**：FDA 批准可使股价翻倍，拒绝可使股价腰斩
**特殊信号**：
- PDUFA 日期（FDA 目标审批日）→ 提前建仓/平仓时间点
- Phase 3 临床试验数据 → 最高影响力的单一事件
- 专利到期 + 仿制药竞争 → 长期负面压力

### 能源
**最敏感**：原油/天然气价格（直接影响利润）、地缘政治
**周期位置判断**：能源股在油价下行周期估值陷阱很多
**特殊信号**：
- OPEC 产量决策 → 油价最重要的单一驱动
- 飓风季 → 墨西哥湾产能短期中断
- 炼厂利润率（crack spread）→ 下游炼油公司先行指标

## 跨行业传导链

```
常见新闻传导路径：

Fed 加息预期上升
  → 科技成长股估值压缩（立即）
  → REITs 承压（立即）
  → 银行 NIM 受益（中期）
  → 消费贷款违约率上升（滞后 6-12 个月）

AI 算力需求新高
  → NVDA/AMD 直接受益（立即）
  → 台积电/ASML 受益（中期）
  → 电力/数据中心 REITs 受益（中期）
  → 传统软件公司竞争压力（长期）

消费信心下滑
  → 可选消费股承压（立即）
  → 必需消费股相对受益（立即）
  → 零售商库存积压风险（1-2 季度滞后）

供应链中断
  → 依赖单一地区的制造商承压（立即）
  → 有多元化供应链的公司相对受益（中期）
```
