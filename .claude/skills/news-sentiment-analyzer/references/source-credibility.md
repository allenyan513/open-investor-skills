# 新闻来源可信度分级指南

> 不同来源的新闻，可信度和影响力差距极大。
> 先判断来源，再判断内容。

---

## 一级来源（最高可信度，直接影响股价）

### 公司官方文件
| 来源 | 说明 | 获取方式 |
|------|------|---------|
| SEC 8-K 文件 | 公司重大事项法定披露 | edgar.sec.gov |
| 财报新闻稿（Press Release）| 官方业绩公告 | 公司 IR 页面 |
| 电话会议记录 | 管理层直接表述 | 公司官网 / Seeking Alpha |
| 代理声明（Proxy）| 高管薪酬/股权信息 | SEC EDGAR |
| 内部人交易表（Form 4）| 高管买卖记录 | SEC EDGAR（实时） |

**使用原则**：这类来源的内容直接入分析，无需核实。

---

## 二级来源（高可信度，需理解立场）

### 主流财经媒体
| 来源 | 特点 | 注意事项 |
|------|------|---------|
| **WSJ / Bloomberg** | 报道准确，信源可靠 | 付费内容，有独家报道 |
| **Reuters** | 速度快，事实导向 | 较少深度分析 |
| **Financial Times** | 宏观视角强 | 英欧视角偏多 |
| **CNBC** | 速度快，市场情绪导向 | 部分观点为嘉宾个人意见 |
| **Barron's** | 分析深度好 | 有时与市场方向相反（逆向参考）|

### 机构研究报告
| 来源 | 特点 | 注意事项 |
|------|------|---------|
| 投行研究报告 | 专业，数据详细 | 存在利益冲突（承销/交易关系） |
| 独立研究机构（Morningstar等）| 更中立 | 可能滞后 |
| 买方研究（对冲基金）| 通常不公开 | 若流出需判断意图 |

**使用原则**：内容可参考，但需理解来源的潜在立场。
投行报告尤其注意：是否承销该公司股票？目标价调整是否跟风？

## 三级来源（中等可信度，需交叉核实）

### 数字财经媒体
| 来源 | 特点 |
|------|------|
| **Seeking Alpha** | 作者质量参差不齐，需看作者历史 |
| **The Motley Fool** | 偏散户，较多 bias |
| **Investopedia** | 教育性内容好，个股分析一般 |
| **Yahoo Finance** | 信息聚合，来源混杂 |

### 社交媒体
| 来源 | 可信度 | 用途 |
|------|--------|------|
| **X（Twitter）** | 低 | 判断市场情绪，非判断事实 |
| **Reddit / r/investing** | 低 | 散户情绪指标，逆向参考 |
| **LinkedIn** | 中 | 行业人士观点，有时有内部消息 |
| **YouTube 财经频道** | 低-中 | 教育性内容可参考，分析慎用 |

**使用原则**：单独不构成分析依据，只用于情绪判断或寻找线索后再核实。

---

## 四级来源（低可信度，主要作为情绪指标）

### 需要极高警惕的来源类型

```
🚨 匿名社交媒体爆料
   - "内部消息"、"可靠渠道"等表述
   - 无法核实，可能是市场操纵

🚨 做空报告（需区分来源）
   高可信：Hindenburg Research、Muddy Waters、Gotham City
           这些机构历史准确率高，发布报告需认真对待
   低可信：匿名做空报告、小型不知名机构
           可能是 pump-and-dump 或 short-and-distort

🚨 分析师"小道消息"（Whisper Numbers）
   流传在社交媒体的非官方预期，可参考但不可依赖

🚨 散户论坛的"内幕"
   绝大多数为谣言或过时信息
```

## 快速来源评估框架

收到一条新闻后，1 分钟内完成：

```
① 来源级别：[1/2/3/4 级]

② 是否有原始文件支撑？
   有 → 可信度 +1
   无 → 保持警惕

③ 发布时间是否合理？
   美股交易时间内的突发新闻（尤其开盘前后）→ 可能是刻意时机

④ 是否有利益冲突？
   做空方发布利空报告 → 核实具体指控
   持仓方发布利好分析 → 了解其持仓情况

⑤ 能否在 1-2 个独立来源中交叉核实？
   能核实 → 可信
   无法核实 → 降级处理，只作为情绪信号
```
