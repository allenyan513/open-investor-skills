# 技术分析入门 Skill

> ⚠️ 技术分析是辅助工具，不是预测工具。
> 本 Skill 帮助判断"当前价格位置是否合理"，不能预测股价涨跌。
> 任何技术信号都需结合基本面综合判断，不构成投资建议。

---

## 核心理念：技术分析的正确定位

新手最常见的误区是把技术分析当成预测工具。正确的认知是：

```
技术分析能做的                  技术分析不能做的
──────────────────             ──────────────────
描述当前价格位置                预测未来价格方向
识别潜在支撑/压力区              保证突破一定成功
衡量短期超买/超卖程度            判断公司基本面好坏
辅助设置止损位                  替代基本面研究
提示趋势可能反转的区域            给出精确买卖点

正确使用姿势：
  基本面研究决定「买什么」
  技术分析辅助判断「什么时候买/买在哪个位置」
```

---

## 分析框架：五层结构

每次技术分析按以下顺序进行，从大到小、从宏观到微观：

```
第一层：趋势判断      → 当前处于上涨/下跌/横盘趋势？
第二层：关键价位      → 重要支撑位和压力位在哪？
第三层：均线系统      → 多头/空头排列？关键均线位置？
第四层：动量指标      → RSI / MACD 判断超买超卖和趋势强度
第五层：成交量确认    → 成交量是否支撑价格行为？
```

> 五层都指向同一方向 = 信号较强
> 层与层之间相互矛盾 = 信号较弱，建议观望

## 第一层：趋势判断

**核心原则**：趋势是你的朋友，不要逆势操作。

### 趋势定义
```
上升趋势：价格不断创造「更高的高点 + 更高的低点」（Higher High + Higher Low）
下降趋势：价格不断创造「更低的高点 + 更低的低点」（Lower High + Lower Low）
横盘整理：高点和低点在一个区间内震荡，方向不明

判断时间维度（三级趋势）：
  主趋势（Primary）：月线/周线级别，数月到数年
  中趋势（Secondary）：日线级别，数周到数月
  短趋势（Minor）：小时线，数天
  
  原则：先看大趋势，再看小趋势。
  大趋势向上但日线回调 → 回调可能是买入机会
  大趋势向下但日线反弹 → 反弹可能是卖出机会
```

### 趋势线画法
```
上升趋势线：连接两个或以上「更高的低点」，价格在线上方运行
下降趋势线：连接两个或以上「更低的高点」，价格在线下方运行

有效趋势线的三个条件：
  ① 至少经过 2-3 个有效接触点（不是随手一画）
  ② 持续时间越长 / 接触次数越多 = 越重要
  ③ 突破趋势线需要配合成交量放大确认

突破信号评级：
  ★★★ 强：收盘价突破 + 成交量 > 20 日均量的 1.5 倍
  ★★  中：收盘价突破 + 成交量略放大
  ★   弱：收盘价刚刚突破，成交量无明显放大（可能假突破）
```

## 第二层：支撑与压力位

**直觉理解**：支撑位是"地板"（价格跌到这里容易反弹），压力位是"天花板"（价格涨到这里容易回落）。

### 关键价位类型

```
类型              特征                        重要程度
──────────────────────────────────────────────────────
历史高点/低点     曾经发生过激烈买卖的价格区域    ★★★
整数关口          $100, $150, $200 等心理价位    ★★
前期密集成交区    大量筹码换手的价格区间          ★★★
均线支撑/压力     重要均线（20/50/200日均线）     ★★
斐波那契回撤位    38.2% / 50% / 61.8% 回撤      ★★
跳空缺口          开盘跳空留下的"空白"区域        ★★
```

### 支撑变压力 / 压力变支撑
```
重要规律：
  价格有效跌破支撑位 → 该支撑位变成新的压力位
  价格有效突破压力位 → 该压力位变成新的支撑位

这是技术分析中最实用的单一规律。
实战应用：
  股票跌破重要支撑后反弹到原支撑附近 → 卖出时机
  股票突破重要压力后回踩原压力附近 → 买入时机（回踩确认）
```

## 第三层：均线系统

均线是最广泛使用的技术工具，本质是「过去 N 天收盘价的平均值」。

### 四条关键均线

```
均线        含义            主要用途
─────────────────────────────────────────
5 日均线    最近一周均价     短线方向，日内交易参考
20 日均线   约一个月均价     中短期趋势，最常用的均线
50 日均线   约两个半月均价   中期趋势，机构常用
200 日均线  约一年均价       长期趋势，最重要的均线之一
```

### 均线形态判断

**多头排列**（看涨信号）：
```
5日 > 20日 > 50日 > 200日
价格 > 所有均线
含义：短中长期都处于上涨趋势
```

**空头排列**（看跌信号）：
```
5日 < 20日 < 50日 < 200日
价格 < 所有均线
含义：短中长期都处于下跌趋势
```

### 金叉与死叉

```
金叉（Golden Cross）★★★
  定义：50 日均线从下向上穿越 200 日均线
  信号：中长期看涨信号，机构常用触发点
  注意：滞后性强，往往在股价已涨了 10-20% 后才出现

死叉（Death Cross）★★★
  定义：50 日均线从上向下穿越 200 日均线
  信号：中长期看跌信号
  注意：同样滞后，往往股价已跌了一段才确认

实用的短期叉：
  5 日上穿 20 日 = 短期买入信号
  5 日下穿 20 日 = 短期卖出信号
```

### 均线斜率
```
均线斜率（向上/向下/平）和均线之间的距离同样重要：
  均线陡峭向上 + 价格在均线上方 = 强趋势
  均线走平 = 趋势减弱，整理阶段
  价格大幅偏离均线（均线乖离率过大）= 短期可能回归均值
  
均线乖离率 = (当前价格 - 均线价格) / 均线价格 × 100%
  乖离率 > +10% → 短期可能回调（过度偏离）
  乖离率 < -10% → 短期可能反弹
```

## 第四层：动量指标

动量指标衡量价格变化的"速度和力度"，用于判断超买/超卖和趋势强弱。

### RSI（相对强弱指数）

**公式**：RSI = 100 - 100 / (1 + 平均上涨幅度 / 平均下跌幅度)
**范围**：0 - 100，**常用参数**：14 日

```
RSI 区间解读：
  > 70     超买区：短期涨幅过大，可能面临回调
  50 - 70  偏强：上涨趋势较健康
  50       中轴：多空分界线
  30 - 50  偏弱：下跌趋势或整理
  < 30     超卖区：短期跌幅过大，可能出现反弹
```

**RSI 使用的四大要点**：

```
① 超买 ≠ 立刻下跌
  强势上涨股票 RSI 可以在 70-90 维持很长时间
  超买只代表"涨得多了"，不代表"一定要跌"

② 背离信号（最有价值的 RSI 用法）
  顶背离：价格创新高，RSI 未能同步创新高
    → 上涨动力减弱，可能转向（看跌信号 ★★★）
  底背离：价格创新低，RSI 未能同步创新低
    → 下跌动力减弱，可能触底（看涨信号 ★★★）

③ RSI 中轴（50）的意义
  RSI 站稳 50 上方 = 多头占优
  RSI 跌破 50 = 多头优势减弱

④ 结合趋势使用
  上升趋势中：RSI 回调到 40-50 附近 = 买入机会（而非超卖的 30）
  下降趋势中：RSI 反弹到 50-60 附近 = 卖出机会（而非超买的 70）
```

### MACD（指数平滑异同移动平均线）

**组成**：快线（EMA12）- 慢线（EMA26）= MACD 线；MACD 线的 EMA9 = 信号线；两者之差 = 柱状图（Histogram）

**常用参数**：(12, 26, 9)

```
MACD 四大信号：

① MACD 线上穿信号线（柱状图由负转正）= 买入信号
  MACD 线下穿信号线（柱状图由正转负）= 卖出信号
  
  ★ 发生在零轴上方的金叉 > 发生在零轴下方的金叉（更强）
  ★ 发生在零轴下方的死叉 > 发生在零轴上方的死叉（更弱）

② 零轴穿越
  MACD 线从下方穿越零轴 = 中期看涨
  MACD 线从上方穿越零轴 = 中期看跌

③ MACD 背离（与 RSI 背离同等重要）
  顶背离：价格新高，MACD 柱状图未同步新高 = 看跌 ★★★
  底背离：价格新低，MACD 柱状图未同步新低 = 看涨 ★★★

④ 柱状图趋势
  柱状图持续缩短（即使仍为正值）= 上涨动力减弱
  柱状图持续放大（即使仍为负值）= 下跌动力减弱
```

### RSI 与 MACD 组合使用

```
信号强度评级：
  ★★★  RSI 底背离 + MACD 底背离同时出现
  ★★★  RSI 顶背离 + MACD 顶背离同时出现
  ★★   仅 RSI 或仅 MACD 出现背离
  ★    RSI/MACD 金叉死叉（无背离配合）

实战规则：
  两个指标发出相同方向信号 → 信号可信度提升
  两个指标发出相反信号 → 观望，等待方向明确
```

## 第五层：成交量分析

**核心原则**：价格是表象，成交量是本质。成交量是市场参与者"用钱投票"的结果。

```
价量关系解读：

量价齐升  价格上涨 + 成交量放大   ★★★ 上涨有效，趋势健康
量减价升  价格上涨 + 成交量萎缩   ⚠️  上涨动力不足，可能假突破
量增价跌  价格下跌 + 成交量放大   🚨  下跌加速，恐慌抛售
量缩价跌  价格下跌 + 成交量萎缩   中性 下跌无量，可能即将结束

关键场景：
  突破时放量  = 有效突破，可信度高  ★★★
  突破时缩量  = 假突破风险，谨慎   ⚠️
  高位缩量    = 买盘枯竭，注意顶部
  低位缩量    = 卖盘枯竭，可能筑底
```

### 成交量异常信号
```
成交量突然放大到 20 日均量的 3 倍以上：
  上涨时 = 主力入场或散户追涨（需判断背景）
  下跌时 = 恐慌性抛售（有时是底部信号 — "卖到没人卖"）

连续多日缩量整理后放量突破 = ★★★ 强烈买入信号
```

## K 线形态速查

常见的单根和组合 K 线形态：

```
看涨形态：
  锤子线（Hammer）      下影线长，实体小，出现在下跌末端
  启明星（Morning Star）三根 K 线：长阴 + 小实体 + 长阳
  吞没形态（Bullish Engulfing）阳线完全吞没前一根阴线
  
看跌形态：
  射击之星（Shooting Star）上影线长，实体小，出现在上涨末端
  黄昏星（Evening Star）  三根 K 线：长阳 + 小实体 + 长阴
  吞没形态（Bearish Engulfing）阴线完全吞没前一根阳线

中性/整理形态：
  十字星（Doji）          开盘价 ≈ 收盘价，表示多空均衡
  纺锤线（Spinning Top）  上下影线均较长，实体小

注意：K 线形态需结合位置判断（出现在什么价位？）
      同一形态在高位 vs 低位含义截然不同。
```

详细的 K 线图例见 `references/candlestick-patterns.md`

## 常见问题解答

**Q：RSI 超买了，该卖出吗？**
```
不一定。强势股可以在超买区停留很长时间。
更重要的信号是：RSI 出现顶背离（价格新高但 RSI 未新高）
单纯 RSI > 70 不是卖出理由，需结合趋势和基本面。
```

**Q：出现金叉/死叉要立刻操作吗？**
```
不建议立刻。均线指标滞后性很强，
金叉出现时股价往往已经涨了 10-20%，追高风险大。
更好的用法：把金叉作为"大方向确认"，
然后等待价格回调到均线附近再考虑买入。
```

**Q：支撑位跌破后还能买吗？**
```
原支撑位跌破后变成压力位。
买入逻辑需要重建：
  ① 等待价格在新的更低支撑位稳定
  ② 等待价格重新突破原支撑位（确认"压力变支撑"）
  在两者都未发生前，不建议接刀。
```

**Q：技术分析在什么情况下会完全失效？**
```
以下情况技术分析参考价值大幅降低：
  ① 财报发布前后（基本面信息主导）
  ② 重大新闻事件（Fed 决议、监管公告等）
  ③ 极低流动性股票（庄股，技术形态可被操控）
  ④ 市场整体崩溃时（相关性趋近 1，个股形态失效）
这些情况下应切换到 news-sentiment-analyzer 或
stock-fundamental-research Skill。
```
