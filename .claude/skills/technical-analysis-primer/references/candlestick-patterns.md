# K 线形态图示与解读

> 所有形态需结合「出现位置」判断，同一形态在上涨末端 vs 下跌末端含义不同。

---

## 如何读懂一根 K 线

```
         │  ← 上影线（最高价到实体顶部）
        ┌┴┐
        │ │ ← 实体（开盘价到收盘价）
        │ │    阳线（绿/白）= 收盘 > 开盘
        │ │    阴线（红/黑）= 收盘 < 开盘
        └┬┘
         │  ← 下影线（实体底部到最低价）

阳线示例：        阴线示例：
    │                  ┌┐
   ┌┴┐                 ││
   │ │                 ││
   └┬┘                 └┘
    │                   │
```

## 单根 K 线形态

### 看涨形态

**锤子线（Hammer）** — 下跌末端出现 ★★★
```
    │  ← 小上影线或无上影线
   ┌┴┐
   │ │ ← 小实体（阳线更强，阴线也有效）
   └┤
    │
    │  ← 长下影线（至少是实体的 2 倍）
    │

含义：跌到低位后有大量买盘涌入，将价格推回
确认条件：次日出现阳线确认更可靠
注意：若出现在上涨趋势中叫「上吊线」，是看跌信号
```

**倒锤子线（Inverted Hammer）** — 下跌末端出现 ★★
```
    │
    │  ← 长上影线
    │
   ┌┴┐
   │ │ ← 小实体
   └┘
   
含义：买方试图推高价格，虽未成功但显示买盘开始活跃
确认条件：需要次日阳线确认
```

**大阳线（Marubozu）** — 任何位置 ★★
```
   ┌┐
   ││ ← 无上下影线的实体阳线
   ││    开盘即最低，收盘即最高
   └┘
   
含义：全天买方完全主导，力度强
```

---

### 看跌形态

**射击之星（Shooting Star）** — 上涨末端出现 ★★★
```
    │
    │  ← 长上影线（至少是实体的 2 倍）
    │
   ┌┤
   │ │ ← 小实体（阴线更强）
   └┘   无下影线或极短

含义：涨到高位后被大量卖盘压回，买方失去控制
确认条件：次日出现阴线确认
注意：若出现在下跌趋势中叫「倒锤子线」，是看涨信号
```

**大阴线（Bearish Marubozu）** — 任何位置 ★★
```
   ┌┐
   ││ ← 无上下影线的实体阴线
   ││    开盘即最高，收盘即最低
   └┘
   
含义：全天卖方完全主导，抛压强烈
```

---

### 中性/转折形态

**十字星（Doji）** — 需结合位置 ★★
```
    │  ← 上影线
    ┼  ← 开盘 = 收盘（或极接近）
    │  ← 下影线

含义：多空力量均衡，当前趋势可能暂停或反转
  出现在上涨高位 → 看跌信号（顶部十字星）
  出现在下跌低位 → 看涨信号（底部十字星）
  出现在横盘中 → 无特殊含义
```

**长腿十字（Long-legged Doji）** ★
```
    │
    │  ← 长上影线
    ┼
    │  ← 长下影线
    │

含义：极度不确定，多空激烈争夺后平局
通常预示短期方向即将明朗
```

## 双 K 线组合形态

### 看涨组合

**看涨吞没（Bullish Engulfing）** ★★★
```
第一根（阴线）  第二根（阳线）
    ┌┐              ┌──┐
    ││           ←  │  │  阳线实体完全覆盖阴线实体
    ││              │  │
    └┘              └──┘

出现位置：下跌趋势末端
含义：买方完全压制卖方，趋势可能逆转
确认：成交量放大更可靠
```

**看涨孕线（Bullish Harami）** ★★
```
第一根（大阴线）  第二根（小阳线）
  ┌───┐               ┌┐
  │   │           ←   ││  小阳线被大阴线实体包含
  │   │               └┘
  └───┘

出现位置：下跌趋势末端
含义：卖压开始减弱，不确定性上升
```

**平底（Tweezer Bottom）** ★★
```
两根 K 线的最低价完全相同或极接近
通常第一根阴线，第二根阳线

含义：两次触及同一低位都被买盘支撑，支撑位强
```

---

### 看跌组合

**看跌吞没（Bearish Engulfing）** ★★★
```
第一根（阳线）  第二根（阴线）
    ┌──┐            ┌┐
    │  │        ←   ││  阴线实体完全覆盖阳线实体
    │  │            └┘
    └──┘

出现位置：上涨趋势末端
含义：卖方完全压制买方，趋势可能逆转
```

**看跌孕线（Bearish Harami）** ★★
```
第一根（大阳线）  第二根（小阴线）
  ┌───┐               ┌┐
  │   │           ←   ││  小阴线被大阳线实体包含
  │   │               └┘
  └───┘

出现位置：上涨趋势末端
含义：上涨动力开始衰竭
```

## 三 K 线组合形态

### 启明星（Morning Star）★★★ — 最重要的底部反转形态
```
第一根        第二根          第三根
（大阴线）    （小实体/十字）   （大阳线）
  ┌───┐
  │   │          ┌┐           ┌───┐
  │   │      →   ││      →    │   │
  │   │          └┘           │   │
  └───┘                       └───┘

出现位置：下跌趋势末端
第三根阳线收盘价需进入第一根阴线实体内部
含义：下跌动力耗尽，买方开始主导

成交量要求：第三根阳线成交量 > 第一根阴线 = 更强信号
```

### 黄昏星（Evening Star）★★★ — 最重要的顶部反转形态
```
第一根        第二根          第三根
（大阳线）    （小实体/十字）   （大阴线）
                                ┌───┐
  ┌───┐          ┌┐             │   │
  │   │      →   ││      →      │   │
  │   │          └┘             └───┘
  └───┘

出现位置：上涨趋势末端
第三根阴线收盘价需进入第一根阳线实体内部
含义：上涨动力耗尽，卖方开始主导
```

### 三白兵（Three White Soldiers）★★ — 底部反转
```
连续三根阳线，每根开盘价在前一根实体内，
且收盘价不断创新高，上影线极短。

含义：持续稳定的买盘，趋势反转信号
注意：出现在超卖后的低位更有意义
```

### 三黑鸦（Three Black Crows）★★ — 顶部反转
```
连续三根阴线，每根开盘价在前一根实体内，
且收盘价不断创新低，下影线极短。

含义：持续稳定的卖盘，趋势反转信号
```

## 形态使用总结

```
最可靠的信号组合（同时满足）：
  ① 形态出现在关键支撑/压力位附近
  ② 形态出现后成交量放大
  ③ RSI 或 MACD 同步出现背离
  ④ 日线形态得到周线趋势支持

常见陷阱：
  × 在横盘整理中任何形态可信度都低
  × 低成交量时出现的形态不可靠
  × 不要孤立看形态，必须结合趋势位置
  × K 线形态在高流动性大盘股更可靠，
    在低流动性小盘股可能被人为制造
```
