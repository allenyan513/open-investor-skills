# 买入 / 卖出前技术面检查清单

> 每次交易前跑一遍。不需要全部打勾，但勾选越多信号越强。
> 反向使用：买入清单的反面 = 卖出/止损信号。

---

## 买入前检查清单

### A. 趋势确认（大方向对了再谈买入）
- [ ] 周线趋势向上（不逆大趋势操作）
- [ ] 日线趋势向上或刚从下跌趋势反转
- [ ] 价格在 200 日均线上方（长期健康）
- [ ] 价格在 50 日均线上方（中期健康）

### B. 位置合理（在对的价格区间买入）
- [ ] 价格接近支撑位（而非压力位）
- [ ] 价格接近重要均线（回调到均线附近）
- [ ] 价格未大幅偏离均线（乖离率 < 10%）
- [ ] 不是追高买入（股价未在短期内大涨 20%+）

### C. 动量信号（买在上涨动力恢复时）
- [ ] RSI 从超卖区（<40）向上回升
- [ ] RSI 出现底背离（价格新低但 RSI 未新低）
- [ ] MACD 出现金叉 或 柱状图由负转正
- [ ] MACD 出现底背离

### D. 成交量确认（钱在流入）
- [ ] 上涨日成交量 > 下跌日成交量（近 10 日）
- [ ] 突破关键压力时成交量放大
- [ ] 无异常大规模抛售信号

### E. K 线形态（可选加分项）
- [ ] 出现看涨 K 线形态（锤子/启明星/吞没）
- [ ] 形态出现在关键支撑位附近
- [ ] 形态出现后成交量放大确认

### F. 风险设置（每次买入必须明确）
- [ ] 已确定止损位（通常在支撑位下方 3-5%）
- [ ] 已计算风险/回报比（建议 ≥ 1:2）
- [ ] 止损幅度 × 仓位比例 ≤ 总资产的 1-2%

---

## 卖出 / 止损清单

### 止损触发（必须执行，不找借口）
- [ ] 价格有效跌破预设止损位
- [ ] 价格有效跌破 200 日均线（长期趋势破坏）
- [ ] 价格有效跌破重要支撑位且成交量放大

### 获利了结信号（部分或全部）
- [ ] 价格触及上方压力位
- [ ] RSI 进入超买区（>70）且出现顶背离
- [ ] MACD 出现死叉且在高位
- [ ] 出现看跌 K 线形态（射击之星/黄昏星）
- [ ] 价格大幅偏离均线（乖离率 > +15%）

### 趋势反转警告（减仓观察）
- [ ] 50 日均线下穿 200 日均线（死叉形成）
- [ ] 均线由多头排列变为混乱或空头排列
- [ ] 价格跌破 20 日均线 + 成交量放大
- [ ] 连续多日缩量上涨（动力不足）

---

## 快速评分

每个打勾计 1 分，按分数给出操作参考：

**买入清单得分**：
```
A+B+C+D 合计：
  12-16 分：强烈买入信号，可足额建仓
  8-11 分：一般信号，建议半仓观察
  4-7  分：信号偏弱，建议观望
  < 4  分：不建议买入
```

**注意**：F 项（风险设置）是硬性前提，
不管得分多高，未设置止损不应建仓。
