# 个股基本面研究 Skill

## 使用前提示

> 本 Skill 输出的是研究框架与分析辅助，**不构成投资建议**。
> 所有数据请以 SEC 官网、公司 IR 页面为准。投资决策需结合自身风险承受能力。

---

## 研究流程（四步法）

拿到股票代码或公司名后，按以下顺序输出完整报告。
每步骤均先用 web_search 获取最新数据，再进行分析。

---

## 第一步：业务快照

**目标**：30 秒内理解这家公司是做什么的、怎么赚钱的。

需要回答的问题：
- 主营业务是什么？收入来自哪几条线？
- 客户是 ToB 还是 ToC？收入是一次性还是订阅/重复性？
- 公司处于行业的什么位置（龙头 / 挑战者 / 利基市场）？
- 最近 1 年有哪些重大事件（并购、高管变动、产品发布）？

**搜索关键词模板**：
```
[TICKER] business model revenue breakdown 2024
[TICKER] latest earnings highlights
[TICKER] annual report investor relations
```

**输出格式**：
```
【业务快照】
公司：[名称]（[代码]）| 行业：[GICS 分类]
一句话：[用一句话描述核心业务]
收入结构：[分拆各业务线占比]
增长驱动：[列出 2-3 个核心增长引擎]
近期大事：[最近 6 个月重要事件]
```

---

## 第二步：财务质量分析

**目标**：判断公司财务是否健康、盈利是否真实。

读取 `references/financial-metrics.md` 获取各指标的优秀/及格/警戒线标准。

### 2A 盈利能力
| 指标 | 数值 | 评级 | 备注 |
|------|------|------|------|
| 毛利率 (Gross Margin) | | | 与行业均值对比 |
| 营业利润率 (Operating Margin) | | | 看趋势是否扩张 |
| 净利润率 (Net Margin) | | | |
| ROE（净资产收益率） | | | >15% 为优质 |
| ROIC（投入资本回报率）| | | 判断资本配置效率 |

### 2B 成长性
| 指标 | 近 1 年 | 近 3 年 CAGR | 评级 |
|------|---------|-------------|------|
| 营收增速 | | | |
| EPS 增速 | | | |
| 自由现金流增速 | | | |

### 2C 财务健康
| 指标 | 数值 | 评级 |
|------|------|------|
| 净负债率 (Net Debt/EBITDA) | | <2x 健康 |
| 流动比率 (Current Ratio) | | >1.5 安全 |
| 利息覆盖倍数 | | >5x 安全 |
| FCF / Net Income 转化率 | | >80% 为佳 |

### 2D 红旗警示 ⚠️
以下任何一条触发需在报告中高亮标注：
- 应收账款增速 >> 营收增速（可能虚增收入）
- FCF 持续为负但净利润为正（盈利质量差）
- 商誉占总资产 >30%（并购风险）
- 高管大量减持（内部人看空）
- 审计师意见非标准无保留意见

## 第三步：估值分析

**目标**：判断当前股价是贵了还是便宜，给出合理价值区间。

读取 `references/valuation-methods.md` 获取各方法的详细计算步骤。

### 3A 相对估值（快速判断）
对比行业均值与历史均值：

| 估值指标 | 当前值 | 行业均值 | 历史均值(5Y) | 判断 |
|---------|--------|---------|-------------|------|
| P/E (TTM) | | | | 贵/合理/便宜 |
| Forward P/E | | | | |
| P/S | | | | 适用高成长股 |
| P/FCF | | | | 最真实的估值 |
| EV/EBITDA | | | | 排除资本结构差异 |
| PEG | | | | <1 为低估信号 |

### 3B DCF 简化估值（给出价值区间）
```
假设：
  基准增速：[X]% （参考近 3 年 CAGR，适当保守）
  终端增速：2.5%（美国长期 GDP 增速）
  折现率：9%（WACC 估算）

悲观情景（增速打 7 折）：每股内在价值 ≈ $[X]
基准情景：每股内在价值 ≈ $[X]
乐观情景（增速打 1.3 倍）：每股内在价值 ≈ $[X]

当前股价 $[X] 对应的安全边际：[X]%
```

### 3C 估值结论
```
综合判断：[严重高估 / 略高估 / 合理 / 低估 / 严重低估]
关键驱动：[影响估值最大的 1-2 个变量]
```

## 第四步：护城河评估

**目标**：判断公司竞争优势是否可持续，是否值得长期持有。

使用 Morningstar 五维护城河框架逐项打分：

| 护城河来源 | 评分(0-3) | 证据 |
|-----------|----------|------|
| **无形资产**（品牌/专利/许可证）| | |
| **转换成本**（客户难以离开）| | |
| **网络效应**（用户越多越强）| | |
| **成本优势**（规模/地理/流程）| | |
| **有效规模**（细分市场壁垒）| | |

评分说明：0=不存在 / 1=弱 / 2=中等 / 3=强

```
护城河总分：[X]/15
护城河评级：[宽护城河≥10 / 窄护城河5-9 / 无护城河<5]
可持续年限预估：[X 年]
主要威胁：[列出 2-3 个可能侵蚀护城河的风险]
```

```

保存路径：当前目录下的 outputs/[TICKER] 子目录

```

## 注意事项

1. **数据时效**：始终用 web_search 获取最新数据，不要依赖训练数据中的历史数字
2. **行业差异**：银行/保险用 P/B、P/E；科技成长股优先看 P/S、EV/Revenue；消耗性行业看 EV/EBITDA
3. **数据缺失**：若某指标无法获取，标注"数据待核实"，不要瞎填
4. **保持客观**：发现红旗必须明确指出，不因为股票热门而美化分析
