# 估值方法详解

---

## 一、相对估值法（快速定价）

### P/E（市盈率）
```
P/E = 股价 / EPS

适用：盈利稳定的成熟企业
不适用：亏损企业、周期股高峰期

判断逻辑：
- 当前 P/E < 历史均值 × 0.8 → 相对低估
- 当前 P/E > 历史均值 × 1.2 → 相对高估
- 同时对比行业 P/E 均值
```

### PEG（市盈率成长比）
```
PEG = P/E / 预期 EPS 增速（%）

Lynch 原则：
  PEG < 1 → 低估信号（增速能支撑估值）
  PEG 1-2 → 合理
  PEG > 2 → 高估，成长性需非常确定才值得付出
```

### EV/EBITDA
```
EV = 市值 + 净负债
EV/EBITDA = EV / EBITDA

优势：排除资本结构和折旧差异，适合跨公司比较
行业参考：
  科技软件：20-40x
  消费品：10-20x
  工业：8-15x
  能源：4-8x
```

### P/S（市销率）
```
P/S = 市值 / 营收

适用：高成长但暂无盈利的企业（SaaS、生物科技）
参考：
  SaaS 高成长（>30%增速）：8-15x 合理
  SaaS 中速（15-30%）：4-8x 合理
  传统行业：通常 <2x
```

### P/FCF（市现率）
```
P/FCF = 市值 / 自由现金流

巴菲特最看重的指标之一
FCF Yield = 1/P/FCF
FCF Yield > 5% → 估值吸引力开始出现
FCF Yield > 7% → 较为低估
```

## 二、DCF 估值法（内在价值）

### 简化三步 DCF

**步骤 1：预测未来 10 年自由现金流**
```python
# 三情景假设
base_fcf = 最近 12 个月 FCF

scenarios = {
  "悲观": base_growth * 0.7,   # 增速打七折
  "基准": base_growth,          # 按近 3 年 CAGR
  "乐观": base_growth * 1.3    # 增速乘 1.3
}

# 前 5 年用假设增速，后 5 年线性收敛至终端增速
terminal_growth = 0.025  # 2.5%，约等于美国长期 GDP
```

**步骤 2：折现（WACC 估算）**
```
保守型公司（低 beta）：折现率用 8%
一般成长股：折现率用 9-10%
高风险成长股：折现率用 10-12%

PV = FCF_year_n / (1 + WACC)^n
```

**步骤 3：终端价值**
```
Terminal Value = FCF_year10 × (1 + g) / (WACC - g)
TV_PV = Terminal Value / (1 + WACC)^10

内在价值 = Σ(各年 FCF 现值) + TV_PV
每股内在价值 = 内在价值 / 流通股数
```

### DCF 敏感性矩阵
计算不同增速和折现率组合下的每股价值：

```
          折现率 8%   折现率 9%   折现率 10%
增速 20%   $XXX        $XXX        $XXX
增速 15%   $XXX        $XXX        $XXX
增速 10%   $XXX        $XXX        $XXX
```

取中间区域作为合理价值区间。

## 三、安全边际原则

Graham / 巴菲特原则：
- 理想买入价 = 内在价值 × 0.7（留 30% 安全边际）
- 成长股可接受：内在价值 × 0.85
- 确定性极高的优质企业：内在价值 × 0.9

```
安全边际 = (内在价值 - 当前股价) / 内在价值 × 100%
```

## 四、常见估值误区

1. **只看 P/E**：忽略成长性、资产质量、现金流
2. **用峰值利润估值**：周期股在景气顶端 P/E 最低，陷阱
3. **DCF 过度精确**：预测 10 年后的数字本身有极大不确定性，结论是区间而非精确值
4. **忽视稀释**：股票数量持续增加会摊薄每股价值，需关注 diluted shares 变化
5. **忽视现金 / 负债调整**：EV 才是真实"购买价格"
