# 财务指标评分标准参考

> 以下为通用标准，实际分析时需结合行业特性调整。
> 科技股、消费股、金融股的合理区间差异较大，见文末行业差异说明。

---

## 盈利能力指标

### 毛利率 (Gross Margin)
| 评级 | 标准 | 含义 |
|------|------|------|
| ⭐⭐⭐ 优秀 | >60% | 强定价权，软件/品牌消费品典型 |
| ⭐⭐ 良好 | 40-60% | 健康，大多数优质企业 |
| ⭐ 一般 | 20-40% | 竞争激烈或重资产行业 |
| ⚠️ 警戒 | <20% | 低利润行业（零售/制造），需看净利率 |

### ROE（净资产收益率）
| 评级 | 标准 | 备注 |
|------|------|------|
| ⭐⭐⭐ 优秀 | >20% | 巴菲特最低门槛是连续 10 年 >15% |
| ⭐⭐ 良好 | 15-20% | |
| ⭐ 一般 | 8-15% | |
| ⚠️ 警戒 | <8% | 低于无风险利率，资本配置效率差 |

注意：ROE 可能被高杠杆虚增，需同时看负债率。

### ROIC（投入资本回报率）
| 评级 | 标准 |
|------|------|
| ⭐⭐⭐ 优秀 | >15% |
| ⭐⭐ 良好 | 10-15% |
| ⭐ 及格 | WACC 以上 |
| ⚠️ 警戒 | < WACC（在摧毁价值）|

### FCF Margin（自由现金流利润率）
| 评级 | 标准 |
|------|------|
| ⭐⭐⭐ 优秀 | >20% |
| ⭐⭐ 良好 | 10-20% |
| ⭐ 一般 | 5-10% |
| ⚠️ 警戒 | <0%（烧钱阶段需看成长性补偿） |

## 成长性指标

### 营收增速
| 评级 | 标准（YoY） | 适用阶段 |
|------|------------|---------|
| 超高成长 | >30% | 早期高成长，估值溢价合理 |
| 高成长 | 15-30% | 成长期，值得关注 |
| 稳健成长 | 5-15% | 成熟期优质企业 |
| 低增长 | 0-5% | 成熟/防御型，靠股息回报 |
| ⚠️ 警戒 | <0% | 收缩，需研究原因 |

### EPS 增速 vs 营收增速
- EPS 增速 > 营收增速：利润率扩张，经营杠杆良好 ✅
- EPS 增速 ≈ 营收增速：稳定 ✅
- EPS 增速 < 营收增速：利润率收缩，需警惕 ⚠️
- 营收增长但 EPS 下降：成本失控或大量稀释 🚨

## 财务健康指标

### 净负债率 (Net Debt / EBITDA)
| 评级 | 标准 |
|------|------|
| ⭐⭐⭐ 净现金 | <0（现金 > 负债） |
| ⭐⭐ 健康 | 0-2x |
| ⭐ 可接受 | 2-3x |
| ⚠️ 偏高 | 3-4x |
| 🚨 危险 | >4x（利率上升时风险极大） |

### 流动比率 (Current Ratio)
| 评级 | 标准 |
|------|------|
| 安全 | >2 |
| 健康 | 1.5-2 |
| 勉强 | 1-1.5 |
| 🚨 危险 | <1（短期偿债压力）|

### FCF / Net Income 转化率
| 评级 | 标准 | 含义 |
|------|------|------|
| ⭐⭐⭐ 优质 | >100% | 利润质量极高 |
| ⭐⭐ 良好 | 80-100% | 健康 |
| ⭐ 一般 | 50-80% | 有一定应收账款问题 |
| ⚠️ 警戒 | <50% | 净利润可能虚高 |

## 行业差异说明

### 科技/SaaS
- 毛利率基准提高：>70% 优秀，60-70% 良好
- 早期可接受 FCF 为负，但需看 Rule of 40（增速 + FCF Margin > 40%）
- 重点关注：净收入留存率 (NRR)、ARR 增速、CAC/LTV

### 银行/金融
- 不适用负债率、流动比率等常规指标
- 改用：NIM（净息差）、NPL（不良贷款率）、CET1 资本充足率、ROA
- ROA > 1% 为良好，ROE > 12% 为优秀

### 零售/消费
- 毛利率标准降低：>30% 良好
- 重点关注：同店销售增速（SSS）、库存周转天数、门店经济模型

### 能源/原材料
- 利润高度周期性，不适合用单一年份 P/E
- 改用 EV/资源储量、周期平均利润估值

### 医疗/生物科技
- 早期无盈利正常，看 pipeline 价值和 TAM
- 商业化阶段重点看药品定价权和专利保护期

### REITs
- 改用 FFO（运营资金流）代替净利润
- 重点关注：出租率、债务到期结构、派息覆盖率
